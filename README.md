# Project for Turkic Game Jam 2025

## Description
This project is developed for the Turkic Game Jam 2025. It is built using Unity and leverages the Universal Render Pipeline (URP) for enhanced graphics and performance. The project includes the necessary settings and assets to get started with URP.

## Plugins/3d party assets  Used

- [Odin Inspector](https://assetstore.unity.com/packages/tools/utilities/odin-inspector-and-serializer-89041)
- [Nova UI](https://assetstore.unity.com/packages/tools/gui/nova-free-trial-231156)
- [DOTween](https://assetstore.unity.com/packages/tools/visual-scripting/dotween-hotween-v2-27676)
- [Rainbow Folders](https://assetstore.unity.com/packages/tools/utilities/rainbow-folders-2-143526)
- [UniTask](https://gisdthub.com/Cysharp/UniTask)
